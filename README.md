# Simple HTTP Server in Go


## Installation

```bash
git clone [repository-url]
cd [repository-name]
go build
```

## Features
 - Serve static files
 - Logs
 - Responds to GET requests
 - against directory traversal attacks.

## Run

```bash
go build
./httpserver_go-main [port]
```
